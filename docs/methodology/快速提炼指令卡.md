# 洞察提炼快速指令卡

> 用于快速提炼文案的Prompt模板，可直接使用

---

## 🚀 基础提炼指令

```
请帮我分析这段文案，提炼出结构化的洞察函数：

【文案内容】
[粘贴文案]

请按以下维度进行提炼：

1. 核心观点（一句话总结）
2. 人格特质（3-5个关键词）
3. 思维模型（2-4个核心模型，含说明）
4. 语言特征：
   - 句式模板（3-5种）
   - 措辞风格（力度、基调、修辞）
   - 高频词汇
5. 洞察流程（思考步骤拆解）
6. 质量标准（必须具备的特征）
7. 禁忌清单（明确拒绝的东西）

最后输出为Lisp风格的函数代码。
```

## 🎯 进阶提炼指令

```
请深度分析这段文案，提炼完整的洞察生成系统：

【原文】
[粘贴文案]

请完成以下任务：

第一步：文本解构
- 标注所有对立关系（A vs B）
- 提取3-5个最具特色的句子
- 统计高频词汇（出现3次以上）

第二步：思维建模
- 识别核心信念
- 分析处理矛盾的方式
- 总结价值判断标准
- 命名2-4个思维模型

第三步：语言解码
- 归纳句式模板（至少5种）
- 定位措辞风格（力度、基调）
- 收集标志性短语

第四步：流程还原
- 倒推作者的思考路径
- 拆解为可执行步骤
- 为每步命名

第五步：标准制定
- 明确质量检验标准
- 列出禁忌清单

第六步：代码生成
- 输出完整的Lisp函数
- 包含主函数和辅助函数
- 附带使用示例

请确保提炼出的系统能够：
✓ 复现原作者的思维方式
✓ 保留标志性语言风格
✓ 可用于生成类似风格的新内容
```

## 🔄 对比提炼指令

```
请对比分析以下两段文案，提炼出它们的共性和差异：

【文案A】
[粘贴文案A]

【文案B】
[粘贴文案B]

请分析：

1. 共性元素（稳定的底层模式）：
   - 共同的思维方式
   - 共同的价值观
   - 共同的语言习惯

2. 差异元素（风格变化）：
   - 措辞力度差异
   - 句式使用差异
   - 话题侧重差异

3. 核心提炼：
   基于共性元素，输出一个通用的洞察函数框架。

4. 变体设计：
   基于差异元素，设计可调节的参数（如力度、场景）。
```

## 🎨 风格迁移指令

```
我想将【作者A】的风格迁移到【作者B】的内容上。

【作者A的文案】
[粘贴A的文案]

【作者B的文案】
[粘贴B的文案]

请完成：

1. 分别提炼A和B的核心特征
2. 识别两者的兼容点和冲突点
3. 设计融合方案：
   - 保留A的哪些要素？
   - 保留B的哪些要素？
   - 如何协调冲突？
4. 输出融合后的洞察函数
5. 给出一个融合示例

目标：创造一个既有A的【特点】又有B的【特点】的新风格。
```

## 🧪 测试验证指令

```
我已提炼出这个洞察函数，请帮我验证它的质量：

【提炼的函数】
[粘贴你的Lisp代码]

【原始文案】
[粘贴原文]

请检验：

1. 完整性检查：
   - 是否捕捉到核心思维模式？
   - 是否保留语言风格？
   - 是否遗漏重要元素？

2. 可用性测试：
   用这个函数生成3个不同话题的洞察，看是否：
   - 符合原作者风格
   - 保持一致的逻辑
   - 可操作性强

3. 改进建议：
   指出需要优化的地方，给出具体修改方案。
```

## 📚 批量提炼指令

```
我有同一作者的多篇文案，想提炼出最稳定的模式：

【文案1】
[粘贴]

【文案2】
[粘贴]

【文案3】
[粘贴]

请完成：

1. 交叉验证：
   - 找出所有文案共有的元素
   - 标注只在部分文案出现的元素
   - 识别随时间变化的趋势

2. 核心提炼：
   - 只提炼出现在所有文案中的稳定模式
   - 将偶发要素标注为"可选项"

3. 版本演化：
   - 如果有明显的风格演变，说明变化方向
   - 建议是提炼早期版本还是最新版本

4. 输出：
   一个更通用、更稳定的洞察函数。
```

## ⚡ 极简提炼指令

> 用于快速提炼，适合时间紧张的场景

```
快速提炼这段文案的核心框架：

[粘贴文案]

只需输出：
1. 核心信念（1句话）
2. 思维模型（2个）
3. 句式特征（3种）
4. 禁忌清单（3条）
5. 简化版Lisp函数（只包含基础结构）
```

## 🎓 学习提炼指令

> 用于学习目的，带详细解释

```
我想学习如何提炼洞察函数，请一边分析一边教我：

【文案】
[粘贴文案]

请在分析时：
1. 每做一个判断，都解释为什么这样判断
2. 展示你的思考过程（不只是结果）
3. 指出容易犯的错误
4. 给出优化建议

最后输出：
- 提炼结果（函数代码）
- 提炼过程笔记（你是如何一步步得出结论的）
- 给我的学习建议（下次我自己提炼时要注意什么）
```

## 💼 商业化提炼指令

> 适合为品牌/个人IP打造专属风格

```
我想为【品牌/个人名称】打造一套专属的洞察生成系统：

【现有文案样本】
[粘贴1-3篇代表性文案]

【品牌定位】
目标受众：
核心价值观：
希望传达的感觉：

请设计：

1. 品牌化的洞察函数：
   - 符合品牌调性
   - 可复用于不同场景
   - 具有辨识度

2. 使用指南：
   - 适用场景
   - 不适用场景
   - 调节参数（如力度、正式度）

3. 内容模板库：
   提供5-10个可直接套用的模板。

4. 禁忌清单：
   列出可能伤害品牌形象的内容。
```

## 使用建议

1. **首次使用**：推荐从"基础提炼指令"开始
2. **需要深度分析**：使用"进阶提炼指令"
3. **验证质量**：用"测试验证指令"检查
4. **时间有限**：用"极简提炼指令"快速完成

可以复制指令后修改其中的具体要求，使其更符合你的需求。
